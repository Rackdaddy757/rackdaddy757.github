Table Cheat Sheet Section

Listed below are different elements that are relavent to creating a table in HTML5.

Tag Name Description
<table> Table The wrapper element for HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows actually containing table data.
<tr> Table Row The table row container
<tfoot> Table Foot The set of rows defining the footer of a table.
